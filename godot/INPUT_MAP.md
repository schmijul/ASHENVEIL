# Ashenveil Input Map

The bootstrap registers these actions at runtime if they are missing from the project settings.

## Movement

- `move_forward` - `W`, `Up`
- `move_backward` - `S`, `Down`
- `move_left` - `A`, `Left`
- `move_right` - `D`, `Right`
- `sprint` - `Shift`

## Combat

- `attack_light` - `Mouse Left`
- `attack_heavy` - `Mouse Right`
- `block` - `Mouse Right` hold
- `dodge` - `Space`

## Interaction

- `interact` - `E`
- `inventory` - `I`, `Tab`
- `quest_log` - `J`
- `pause` - `Escape`

## Camera

- `camera_reset` - `R`
